# Changelog

All notable changes to the TeamOfThePing_F2025 modpack will be documented in this file.

## [1.2.0] - 2026-01-11
### Added
- Added `WindAtYourBack` as a replacement for `OdinShipPlus` windbelts.
- Added `AdjustableBreedingLimit` to remove the requirement for building unsightly breeding towers.
- Added `ValheimModding-JsonDotNET` as a dependency.
- Added `ConfigurationManage` for simplified config editing in-game.

## [1.1.0] - 2026-01-07
### Added
- Added `ValheimRAFT` to allow for custom-built modular ships.
- Added `balrond_shipyard` for advanced naval docking and construction.
- Added `ExtraSlots` to provide dedicated equipment and quick-use slots.
- Added `VNEI` (Valheim Networked Extensive Info) for in-game recipe and item browsing.
- Added `Mining` skill by Smoothbrain.
- Added `YamlDotNet` as a required library dependency.

### Changed
- Refactored dependencies for better compatibility with Pterodactyl server environments.

### Removed
- Removed `OdinShipPlus` in favor of the more customizable RAFT building system.

## [1.0.1] - 2025-12-29
### Changed
- Refactored `README.md` for better clarity.
- Updated metadata to include the official GitHub repository link.

## [1.0.0] - 2025-12-28
### Added
- Initial release for the TeamOfThePing server.
- Core QoL suite (Quick Stack, MultiUserChest, AzuCraftyBoxes).
- Base skill systems (Sailing, Farming, Blacksmithing, Cooking).
- Agriculture and Building expansions (PlantEverything, OdinArchitect).
